# Assets Directory

This directory contains static assets for the 3D Perspective Viewer app.

## Required Assets

Place the following files in this directory:

### Icons
- `icon.png` - App icon (1024x1024 recommended)
- `adaptive-icon.png` - Adaptive icon (1024x1024)
- `favicon.png` - Web favicon (32x32)

### Splash Screen
- `splash.png` - Splash screen image (1284x2778 recommended)

## Placeholder Assets

For development, Expo will generate default assets if these are missing.

To generate custom icons:
```bash
npx expo-optimize assets/
```

To generate app icons:
```bash
npx @expo/icon-builder generate
```
